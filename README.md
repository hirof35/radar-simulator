# Phased Array Radar Simulator

TypeScript と HTML5 Canvas を使用した、フェーズドアレイレーダー（位相配列レーダー）の電波干渉・ビームフォーミング（波束形成）をリアルタイムに計算する視覚シミュレーターです。

物理的にアンテナを動かすことなく、各素子から放射される電波の「位相（タイミング）」をずらすだけで、電波のビーム（メインローブ）を特定の方向へ瞬時に向けられる仕組みを直感的に学べます。

## 💡 特徴

- **波の重ね合わせの原理をリアルタイム計算**: Canvas上の各ピクセルにおいて、すべてのアンテナ素子から届く波の振幅をリアルタイムに合算・描画。
- **インタラクティブな制御**: スライダーを動かすことで、ビームの操舵角（向き）やアンテナの素子数を即座に変更可能。
- **高速な開発環境**: 超高速なフロントエンドビルドツール `Vite` を採用し、TypeScript を設定なしで即座にブラウザ実行。

## 🛠 動作原理（数理背景）

隣り合うアンテナ素子間の距離を $d$、電波の波長を $\lambda$ としたとき、ビームを狙った角度 $\theta$ に向けるための素子間の位相差 $\Delta \phi$ は、以下の数式に基づいて計算されています。

$$\Delta \phi = \frac{2\pi}{\lambda} \cdot d \cdot \sin(\theta)$$

- **素子数（Element Count）**: 増やすほど干渉の密度が上がり、メインビームがより鋭く（細く）なり、探知精度が向上します。
- **操舵角（Steering Angle）**: 各素子の位相をこの数式で一括制御することで、機械的な首振りをせずに電波の指向性を制御（電子スキャン）します。

## 📂 フォルダ構成

```text
radar-simulator/
├── index.html          # UI（スライダー、Canvas）の配置
├── tsconfig.json       # TypeScript コンパイル設定
├── package.json        # 依存ライブラリ・スクリプト定義
└── src/
    └── main.ts         # シミュレーター本体・DOM連携ロジック
🚀 動かし方
前提条件
パソコンに Node.js がインストールされている必要があります。

1. 依存ライブラリのインストール
プロジェクトのルートフォルダでターミナルを開き、以下のコマンドを実行します。

Bash
npm install
(※手動で構築した場合は npm install -D vite typescript を実行してください)

2. 開発サーバーの起動
以下のコマンドでローカル開発サーバーを起動します。

Bash
npx vite
実行後、ターミナルに表示されるURL（例: http://localhost:5173/）にブラウザでアクセスすると、シミュレーターが動作します。
